# CurrencyCalculator

아이폰용 환율 계산기 앱입니다.

## 적용 가능 화폐 단위

다음 화폐 단위들로 환산 가능합니다. (다른 화폐단위끼리 환산은 불가능합니다.)

 * 호주 달러 (AUD)
 * 브라질 헤알 (BRL)
 * 캐나다 달러 (CAD)
 * 스위스 프랑 (CHF)
 * 중국 위안 (CNY)
 * 유럽 유로 (EUR)
 * 영국 파운드 (GBP)
 * 홍콩 달러 (HKD)
 * 인도 루피 (INR)
 * 일본 엔 (JPY)
 * 멕시코 페소 (MXN)
 * 러시아 루블 (RUB)
 * 태국 바트 (THB)
 * 대만 달러 (TWD)
 * 미국 달러 (USD)
 * 베트남 동 (VND)
